## Description

This project is a Nest backend integrated with postgres/postgis extension. It aims to offer a way to process geospatial data within usage of methods provided from postgis extension.
## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
